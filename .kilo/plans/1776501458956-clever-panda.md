# Fix: Instructions popup missing X button and close functionality

## Problem
1. The X button added to the craft-popup div is not showing up
2. When `showInstructions()` is called, it uses `.textContent = formatInstructions(data)` which overwrites the entire div content, including the X button
3. Need a proper close mechanism

## Solution
1. Wrap the content in the popup div with an inner container, so the X button isn't overwritten
2. Use innerHTML instead of textContent, or target the inner content specifically
3. Add a close button that isn't affected by the content update

1. Modify the craft-popup HTML to have an inner content wrapper
2. Update showInstructions() to target only the inner content, not the whole popup
