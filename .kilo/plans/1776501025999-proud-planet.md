## 1. Add KAT Flips Feature

### 1.1 Backend - Netlify Function (`netlify/functions/api.js`)

**New API**: `https://sky.coflnet.com/api/kat/data`
- Fetch pet upgrade data (COMMON ŌåÆ UNCOMMON ŌåÆ RARE ŌåÆ EPIC ŌåÆ LEGENDARY)
- The API returns pet upgrade recipes with `cost`, `material`, `hours`, `materials` fields
- Need to fetch pet sell prices from Bazaar to calculate actual profits

**Implementation in Netlify Function**:
- Add `fetchKatFlips()` function that fetches KAT data
- Cache KAT data for 5 minutes (pet data changes rarely)
- Merge into existing `/api/flips` response under `katFlips` key

**Note**: Since there's no permanent server, all API calls go through Netlify Functions which act as a proxy/edge function.

### 1.3 KAT Profit Calculation

**Good news!** There's already a `/api/kat/profit` endpoint that provides:
- `profit` - calculated profit (already done for us!)
- `volume` - daily transaction count
- `median` - median sell price at target rarity
- `purchaseCost` - cost to buy the base pet
- `materialCost` - cost of upgrade materials
- `originAuctionName` - starting pet (e.g., "[Lvl 1] Pig")
- `targetRarity` - target rarity (LEGENDARY, EPIC, etc.)
- `coreData` - pet details (name, baseRarity, hours, materials)

**API Response format**:
```javascript
{
  productId: "PET_PIG",
  name: "Pig",
  fromRarity: "EPIC",
  toRarity: "LEGENDARY",
  purchaseCost: 2000000,
  materialCost: 6284687.7,
  median: 10445999,
  profit: 2215312.3,
  volume: 35,
  hours: 0.0,
  materials: {"GIANT_PIG_EAR": 1}
}
```

## 2. Site Optimization

### 2.1 Performance - Add Better Caching

**Problem**: Currently 55-second cache, but API calls could be optimized.

**Solution**:
- Add in-memory LRU cache in Netlify function
- Cache KAT data separately (changes less frequently than flips)
- Add `If-Modified-Since` header support to skip unchanged responses
- Implement stale-while-revalidate pattern for faster perceived latency

**Caching strategy**:
| Data Type | Cache Duration | Notes |
|----------|--------------|-------|
| Bazaar Flips | 55s | Keep existing |
| Craft Flips | 55s + per-user | User-specific caching |
