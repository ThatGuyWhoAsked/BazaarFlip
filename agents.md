# Agents

This project currently utilizes a single primary agent, Gemini CLI, to manage development tasks and interactions.

## Gemini CLI (Primary Agent)
- **Role**: Interactive CLI agent specializing in software engineering tasks, codebase analysis, and project modifications.
- **Capabilities**:
    - File system interaction (read, write, search, list directories).
    - Shell command execution.
    - Web searching.
    - Code modification and refactoring.
    - Contextual understanding and problem-solving.
- **Contribution**: Gemini CLI is responsible for understanding user requests, researching the codebase, formulating strategies, and executing changes to fulfill the project's requirements.
